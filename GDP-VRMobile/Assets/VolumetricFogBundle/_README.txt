
This asset contains 2 main packages. Please use the appropriate package according to the rendering pipeline of your project:

- Volumetric Fog & Mist 2 for Universal Rendering Pipeline (URP) -> folder /URP.
Requires:
  1) Unity 2019.3 or later.
  2) Universal_RP 7.1.8 (or later) installed from the Package Manager.
Once imported, please refer to the documentation (PDF) included for additional setup instructions.


- Volumetric and Dynamic Fog & Mist for Builtin Pipeline -> folder /Builtin.
Requires Unity 2018.3 or later.
Just import the package and read the documentation (PDF) for details.
